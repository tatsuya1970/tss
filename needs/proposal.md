上記3つの課題に対し、**2つのAIシステム**を組み合わせたアプローチをご提案します。

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  ① 広島市町情報ダッシュボード                      │
│     全23市町 公式サイト → AI要約・分類 → 一覧表示   │
│                                                 │
│  ② SNSトレンド可視化ツール                        │
│     X（リアルタイム） + Instagram（準備済み）      │

└─────────────────────────────────────────────────┘
```

両システムとも**稼働するプロトタイプが存在**しており、デモが可能です。

## 3. ソリューション① 広島市町情報ダッシュボード

### 概要
広島県内**全23市町の公式サイト**をAIが自動巡回し、新着情報を収集・要約・分類してダッシュボードに表示します。

### 対応市町（全23）
広島市、福山市、呉市、廿日市市、尾道市、東広島市、竹原市、三原市、府中市、三次市、
庄原市、大竹市、安芸高田市、江田島市、府中町、海田町、熊野町、坂町、安芸太田町、
北広島町、大崎上島町、世羅町、神石高原町

### 主な機能
- **自動収集**：各市町の新着情報・プレスリリースページを定期スクレイピング
- **AI分類**：「防災・安全」「福祉・医療」「経済・産業」「教育・文化」「インフラ・都市」に自動分類
- **AI要約**：タイトルとニュースバリュースコア（1〜5）を自動付与
- **フィルター機能**：市町・カテゴリ・期間で絞り込み表示
- **差分管理**：前回取得以降の新着のみを検出・保存

### 導入効果
- リサーチャーが23市町を手動確認する工数を**大幅削減**
- カテゴリ・スコアにより**重要情報を見逃さない**仕組み
- 過去データの蓄積により、地域トレンドの**時系列分析**が可能

### 今後の拡張ロードマップ
| フェーズ | 内容 |
|---------|------|
| Phase 2 | 広報資料（PDF等）の本文取得と詳細要約 |
| Phase 2 | オープンデータポータルとの連携 |
| Phase 3 | 記事本文全文を読んだ高精度な要約・構造化 |

## 4. ソリューション② SNSトレンド可視化ツール

### 概要
SNSの投稿をリアルタイムに収集し、広島県内のどの地域で何が盛り上がっているかを**地図上のヒートマップ**で可視化します。

### X（旧Twitter）連携の価値

> **「公式発表より早く、現場の声が上がってくる」**

情報量は多くないものの、Xのリアルタイム投稿には他のメディアでは代替できない独自の価値があります。

- イベント当日の**混雑・行列・盛り上がり**をその場で把握
- 災害・事故・交通渋滞などの**速報情報**をいち早くキャッチ
- テレビが気づいていない**ローカルネタの発掘**に活用

### 主な機能
- **地図ヒートマップ**：投稿の熱量を地図上で直感的に把握
- **盛り上がりスコアリング**：投稿数・いいね・リポスト数から「熱量」を算出
- **AIフィルタリング**：他県の同名地名（北海道の北広島、東京の府中等）を自動除外
- **カテゴリ分類**：イベント・グルメ・交通トラブル等を自動ラベリング
- **投稿詳細表示**：ピンをクリックで元投稿を確認可能

### SNS拡張ロードマップ
| プラットフォーム | 状態 | 特徴 |
|----------------|------|------|
| X（旧Twitter） | **稼働中** | リアルタイム・速報性 |
| Instagram | **準備済み**（API取得次第即連携） | グルメ・観光・イベントの盛り上がり |
| TikTok | 今後対応予定 | 若年層の地域情報 |
| Google Mapsクチコミ | 今後対応予定 | 飲食・観光スポットのリアルな声 |

> **XとInstagramが揃うことで、「速報性」と「ビジュアル・賑わい情報」の両軸をカバーできます。**

## 5. 統合オプション

現在は2つの独立したシステムとしてご提案していますが、**1つの統合ダッシュボードにまとめることも検討可能**です。

> 例：「公式サイトの新着情報」と「SNSのリアルタイムトレンド」を同一画面で並べて確認できる、広島県版ニュース統合プラットフォーム

導入規模や運用体制に合わせて、分離・統合いずれの形でも対応できますので、ヒアリングの中でご要望をお聞かせください。

## 6. システム概要（技術）

### 広島市町情報ダッシュボード

- **フロントエンド**：React / TypeScript
- **SNS連携**：X API v2 / Instagram Graph API
- **AI**：OpenAI GPT-4o-mini（フィルタリング・分類）
